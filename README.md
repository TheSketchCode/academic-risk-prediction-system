# 🎓 Academic Risk Prediction System

## 📌 Project Overview
The Academic Risk Prediction System is a Machine Learning-based web application designed to identify students who are at risk of poor academic performance.

The system analyzes key factors such as attendance, internal marks, and assignment completion to predict whether a student is likely to be at academic risk. This helps educators take early actions and provide necessary support.

---

## 🚀 Features
- 🔍 Predict student academic risk using Machine Learning
- 📊 Input-based prediction (Attendance, Marks, Assignments)
- 🗂 Store and fetch student data using SQLite database
- 🖥 User-friendly interface built with Streamlit
- 📈 Displays probability of academic risk

---

## 🛠 Technologies Used
- Python
- Scikit-learn (Logistic Regression)
- Streamlit
- SQLite
- Pandas, NumPy

---

## ⚙️ How It Works
1. User enters student details:
   - Attendance (%)
   - Internal Marks (%)
   - Assignment Completion (%)

2. The trained Logistic Regression model processes the data

3. The system predicts:
   - Whether the student is at risk
   - Probability score of risk
